从节点右侧 ● 端口拖到另一节点左侧 ● 端口即可连线。输入框不能接收连线；AI 框可接收多个上游（按顺序拼接），也可向多个下游输出；连线不能成环。滚轮可上下/左右平移画布，或在空白处按住拖动；节点多时点「适配视图」一键缩放看全。

导入文档

粘贴或选择文本文件，将按段落拆分为多段。可选「逐段批处理」：每段自动配一个 AI 框并连好线，一键跑完全篇。

批处理下，每段会生成一个「输入框 → AI 框」微型链。生成后点工具栏「向下游全部运行」即可逐段处理整篇。

API 设置

密钥仅保存在本机浏览器（localStorage），不会上传到任何服务器。可切换为 OpenAI / 通义 / 本地 Ollama 等任意兼容端点。
